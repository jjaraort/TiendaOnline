🛒 Tienda Online (E-Commerce)

Este proyecto es una aplicación de tienda online (e-commerce) desarrollada con una arquitectura por capas en .NET, separando responsabilidades para lograr un sistema escalable, mantenible y organizado.

🧱 Arquitectura del proyecto:
      El sistema está dividido en las siguientes capas:
      CapaEntidad → Modelos de datos (entidades del sistema)
      CapasDatos → Acceso a datos (Base de datos, consultas, repositorios)
      CapaNegocio → Lógica de negocio
      CapaPresentacionAdmin → Panel administrativo
      CapaPresentacionTienda → Interfaz de la tienda para usuarios

🚀 Características:
      Catálogo de productos dinámico
      Carrito de compras
      Gestión de usuarios
      CRUD de productos
      Integración con API backend en C#
      Interfaz responsive con CSS
      Arquitectura modular y escalable

🛠️ Tecnologías utilizadas:

      Frontend:
        JavaScript (61.2%)
        CSS (34.9%)
        HTML (1.4%)
      
      Backend:
        C# (2.2%)
        ASP.NET Core Web API
      
      Herramientas y workflows:
        Deno (testing y ejecución ligera)
        Webpack (bundling del frontend)
        GitHub Actions (CI/CD)
        Datadog Synthetics (testing automatizado)
